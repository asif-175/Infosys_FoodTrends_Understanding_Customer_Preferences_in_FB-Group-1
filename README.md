# Food Trends – Understanding Customer Preferences in FB

## 📌 Project Overview
This project analyzes food trends and customer preferences using data visualization techniques. The objective is to understand purchasing behavior, popular food items, and sales patterns to derive meaningful insights for business decision-making.

## 🎯 Objectives
- Analyze customer purchasing behavior
- Identify top-selling food items
- Understand revenue distribution across categories
- Study time-based sales patterns
- Explore the relationship between price and demand

## 📊 Dataset
The dataset used in this project is **Balaji Fast Food Sales Dataset**, which includes:

- Order ID
- Date
- Item Name
- Item Type
- Item Price
- Quantity
- Transaction Amount
- Transaction Type
- Received By
- Time of Sale

## 🛠 Tools & Technologies
- Power BI
- Excel / CSV Dataset
- Data Visualization Techniques
- DAX (Data Analysis Expressions)

## 📈 Key Analysis & Visualizations
- KPI Cards (Total Revenue, Average Order Value)
- Bar Chart (Revenue by Item)
- Pie Chart (Category Distribution)
- Column Chart (Sales by Time of Sale)
- Scatter Plot (Price vs Quantity)
- Interactive Filters (Slicers)

## 💡 Key Insights
- Fast food items contribute more to total revenue compared to beverages
- Certain items generate higher revenue and are customer favorites
- Peak sales occur during specific times of the day
- Pricing influences customer purchasing behavior

## 🚀 Project Milestones
- Milestone 1: Basic Data Understanding
- Milestone 2: Dashboard Design & Visualization
- Milestone 3: Advanced Analysis
- Milestone 4: Final Insights & Presentation

## 📌 Conclusion
The project successfully identifies customer preferences and food trends, helping businesses make data-driven decisions to improve sales and customer satisfaction.
